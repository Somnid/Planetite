-Use Arrow Keys to move left and right
-use space to jump
-use down to duck
-use ctrl to dig beneath you
-use ctrl + left/right to dig to the left/right
-use Escape to pause access menu
-use Shift to attack

-use the save button to save current world state
-use the load button to load saved world state

You will need to reset upon death

You may add a custom soundtrack by dragging a compatible audio file to the music settings tab.
